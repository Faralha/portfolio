## Features
- Full Authentication System with Role-Based Access Control (Admin, Teachers, Students)
- Teacher Registration using magic link via email
- Classroom Management (Semesters, Course, Class, Enrollments)
- Attendance System using Code
- Tuition and Salary Management
- Automatic Report Generation (PDF) server-side

## Tech Stack
- **Frontend**: Nuxt.js 4 + Nuxt UI + TailwindCSS
- **Backend**: NestJS
- **Database**: PostgreSQL
- **Message Queue**: BullMQ + Redis

## Architecture and Why
- **Docker**: More secure, easier to maintain, zero downtime deployment
- *Microservices*: Separation of concerns, easier to scale and deploy
- **Event-driven Message Queue**: Decoupled services, better performance for tasks like report generation

## CI/CD pipelines
- Automated testing (lint, unit, e2e) on pre-commit and pull requests
- Automated Docker image build on push to main branch
- Automated deployment to production server using Docker Compose
